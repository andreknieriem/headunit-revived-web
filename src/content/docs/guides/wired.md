title: Wired USB Connection
description: Connecting your phone via USB cable.
---

The wired connection is the most stable way to use Headunit Revived.

## Steps to Connect

1. **Connect the devices:** Use a high-quality USB cable to connect your phone to the tablet (headunit).
2. **Select Android Auto:** On your phone, make sure Android Auto is selected in the USB options (Host-Mode if necessary).
3. **Open the app:** Start Headunit Revived on the tablet.
4. **Grant USB Permission:** Click the **USB Button** in the app. A system dialog should appear asking for permission to access the USB device.
5. **Start Projection:** Find your phone in the list, click the button to allow the connection, and wait for the Android Auto interface to appear.

## Troubleshooting USB

- **Cable quality:** Many connection issues are caused by poor-quality cables. Try another one if it fails.
- **USB Hubs:** If you use a USB hub, ensure it is powered or high-quality.
- **Permissions:** If you don't see the USB permission dialog, try unplugging and replugging the cable.
