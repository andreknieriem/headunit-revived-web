---
title: Wireless Launcher
description: Using the Wireless Launcher app with Headunit Revived.
---

Headunit Revived is compatible with the [Wireless Launcher](https://play.google.com/store/apps/details?id=com.borconi.emil.wifilauncherforhur) app, which can help automate the connection process.

## Setup

1. **Enable in Settings:** In Headunit Revived, go to settings and enable **Wireless Headunit Server**.
2. **Network:** Ensure both devices are in the same WiFi network.
3. **Start Launcher:** Open Wireless Launcher on your phone and start it manually (or set it to trigger via Bluetooth).
4. **Auto-Connect:** The Launcher app will find your headunit and send a connection request automatically.

:::note
Wireless Launcher is a third-party app and may require its own configuration.
:::
